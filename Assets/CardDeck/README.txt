# Minimal SVG Card Deck

This package contains an original minimalist 52-card deck in SVG format, plus one optional card back.

Files:
- AS.svg ... KS.svg = spades
- AH.svg ... KH.svg = hearts
- AD.svg ... KD.svg = diamonds
- AC.svg ... KC.svg = clubs
- BACK.svg = optional card back

Canvas size:
- 750 × 1050 px

Notes:
- Face cards are minimalist monogram designs rather than illustrated royals.
- SVGs use generic serif text for rank letters/numbers and custom vector suit shapes.
